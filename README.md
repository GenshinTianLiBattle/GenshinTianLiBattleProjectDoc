# GenshinTianLiBattleProjectDoc

# 项目简介

该项目致力于为原神3.3推出的常驻玩法 【七圣召唤】 提供外置的PVP天梯排行榜，以及自动选取对手进行比赛等功能。

登陆游戏后自动根据uid登录战网账号，然后如果开始匹配，则服务器在线匹配对手，并获取对应uid，客户端将模拟键鼠操作，自动点击游戏界面引导与对手对战，玩家对战后根据画面判断胜负，之后将数据上传到服务器

- [ ] 自动上传战绩
